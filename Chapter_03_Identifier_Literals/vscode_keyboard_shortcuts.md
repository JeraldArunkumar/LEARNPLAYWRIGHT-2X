# VS Code Keyboard Shortcuts

| Action | Windows / Linux | macOS |
|---|---|---|
| Open Command Palette | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Quick Open file | `Ctrl+P` | `Cmd+P` |
| Toggle Terminal | `Ctrl+` ` ` ` | `Ctrl+` ` ` ` or `Cmd+` ` ` ` |
| Open Settings | `Ctrl+,` | `Cmd+,` |
| Find in File | `Ctrl+F` | `Cmd+F` |
| Replace in File | `Ctrl+H` | `Cmd+Option+F` |
| Find in Folder | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| Go to Line | `Ctrl+G` | `Cmd+G` |
| Toggle Sidebar | `Ctrl+B` | `Cmd+B` |
| New File | `Ctrl+N` | `Cmd+N` |
| Save File | `Ctrl+S` | `Cmd+S` |
| Save All | `Ctrl+K S` | `Cmd+Option+S` |
| Close Editor | `Ctrl+W` | `Cmd+W` |
| Split Editor | `Ctrl+\` | `Cmd+\` |
| Format Document | `Shift+Alt+F` | `Shift+Option+F` |
| Comment Line | `Ctrl+/` | `Cmd+/` |
| Block Comment | `Shift+Alt+A` | `Shift+Option+A` |
| Rename Symbol | `F2` | `F2` |
| Go to Definition | `F12` | `F12` |
| Show Problems | `Ctrl+Shift+M` | `Cmd+Shift+M` |
| Toggle Zen Mode | `Ctrl+K Z` | `Cmd+K Z` |
