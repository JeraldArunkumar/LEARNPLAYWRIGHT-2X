# Learn Playwright 2X

This repository contains JavaScript learning examples focused on Playwright and JavaScript fundamentals.

## Repository Structure

- `Chapter_01_Basics/`
  - Basic Playwright and JavaScript examples
- `Chapter_02_Javascript_Concepts/`
  - JavaScript concept examples and identifier rules
- `Chapter_03_Identifier_Literals/`
  - Identifier literal examples and related JavaScript syntax
  - `vscode_keyboard_shortcuts.md` — VS Code shortcut reference

## Notes

- Files in each chapter demonstrate syntax, naming conventions, and valid/invalid identifier usage.
- The repository is intended for learning JavaScript identifiers and coding practices while using VS Code.

## How to Use

1. Open the repository in Visual Studio Code.
2. Review chapter files under each folder.
3. Run or experiment with example scripts in a JavaScript runtime such as Node.js if applicable.
